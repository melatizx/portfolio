Projeto mais recente:

Desempenho de vendas

   Este projeto se trata de um programa onde dispara automaticamente pelo Microsoft Teams o desempenho do vendedor, aumentando a efetividade da equipe e produtividade.

   Para este projeto, foram utilizados Python e Django para o processamento e a manipulação de dados, além de MySQL para armazenamento dos resultados do vendedores.

   Os dados de vendas são obtidos a partir de uma planilha Excel, na qual o programa localiza a linha correspondente com base no nome do vendedor e o envia.

   Por se tratar de um programa que lida com dados sensíveis, não posso disponibilizar a versão original para acesso com os dados SQL, porém o codigo fonte estará disponivel para donwload ou, caso prefira, o link para o repositório Git Hub.
